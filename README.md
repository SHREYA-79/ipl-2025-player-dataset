## Project Overview

This project provides a structured IPL 2025 player dataset created by collecting, validating, cleaning, and organizing player information from publicly available cricket sources.

The dataset was designed for data analytics, visualization, machine learning, SQL practice, and sports analytics applications.

## Dataset Statistics

- 250+ IPL 2025 Players
- 5 Structured Attributes
- Multiple Player Roles
- Active and New Debut Players
- CSV and Excel Formats

## Dataset Schema

| Column | Description |
|----------|-------------|
| player | Player Name |
| ipl_debut | IPL debut year |
| specialization | Batter, Bowler, All-rounder, Wicketkeeper Batter |
| date_of_birth | Date of birth |
| total_no_of_matches_in_ipl | Total IPL matches played |
